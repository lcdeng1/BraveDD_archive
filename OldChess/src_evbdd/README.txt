
Running make should build the following executables.

build_bdd
		Reads a partially-specified function from an input file,
		and builds a BDD representation.  Run build_bdd -h
		for a list of options; some options shown might not be
		implemented.

read_bdd
		Read a BDD from an exchange file, and query it.

read_bdds
		Read one or more BDDs from exchange file(s).

test_parser
		For testing the input file parser.
